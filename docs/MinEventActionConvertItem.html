MinEventActionConvertItem

Triggered effect that will convert the player's holding item into another item, once it's max usage is done.


This allows you to set a durability of an item that breaks to something else.


<triggered_effect trigger="onSelfPrimaryActionEnd" action="ConvertItem, SCore" downgradeItem="meleeClub" maxUsage="10" />
