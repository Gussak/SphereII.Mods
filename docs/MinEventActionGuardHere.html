MinEventActionGuardHere

Sets the guard position to an NPC where the leader or owner is standing.


If there are no leader or owner, for example, if the order was given by a pathing cube, then it sets the guard position to where the entity is standing.


<triggered_effect trigger="onSelfBuffUpdate" action="GuardHere, SCore" />
